# 🪙 Coin Clicker Game

A browser-based clicker game with anti-cheat detection, upgrades, and a leaderboard system.

## Features

### 🎮 Core Gameplay
- **Click to Earn**: Click the main button to earn coins
- **Anti-Cheat Protection**: Only counts legitimate mouse clicks
- **Cheat Detection**: Blocks auto-click attempts and displays warnings

### ⬆️ Upgrade System
- **Click Multiplier** - Double or triple your coins per click
- **Passive Income** - Earn coins automatically every second
- **Auto-Click Income** - Earn coins every 10 seconds
- **Mega Passive** - Earn massive coins per second

Each upgrade increases in cost with each level purchased.

### 🏆 Leaderboard
- **Top 10 Rankings**: Shows the best players
- **Auto-Updates**: Refreshes every 90 seconds
- **Live Tracking**: Your rank updates in real-time
- **Persistent Storage**: Scores are saved locally

### 🛡️ Anti-Cheat System
- **Mouse Button Detection**: Only real clicks count
- **Auto-Click Blocker**: Prevents automated clicking
- **Cheat Counter**: Tracks blocked cheat attempts
- **Visual Warnings**: Shows when cheating is detected

## How to Play

1. **Click the Button**: Click the main 🪙 button to earn coins
2. **Buy Upgrades**: Use coins to purchase upgrades that increase earnings
3. **Progress**: Watch your income grow and climb the leaderboard
4. **Customize**: Set your player name to appear on the leaderboard

## Anti-Cheat Examples

These actions **will be blocked**:
- ❌ Using browser console to modify coins
- ❌ Simulating clicks with JavaScript
- ❌ Using auto-clicker software
- ❌ Clicking without mouse button press

These actions **are allowed**:
- ✅ Clicking the button with your mouse
- ✅ Buying upgrades
- ✅ Letting passive income earn coins
- ✅ Using auto-click upgrade legitimately
